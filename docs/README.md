# Stock Management System - Development Specification

**For Interns: Build this system from scratch**

This specification provides complete requirements to build an identical Stock Management System.

## Quick Start

1. Initialize a Next.js 16 project
2. Configure Tailwind CSS
3. Set up MySQL connection
4. Follow the 5-phase development plan in [SPECIFICATION.md](./SPECIFICATION.md)

## What to Build

A stock management application with:
- User authentication (JWT)
- Monthly stock tracking
- Category hierarchy management
- Purchase records
- Weekly usage tracking
- Read-only protection for old months
