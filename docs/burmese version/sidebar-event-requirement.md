# Sidebar Event Requirements: Category Management Page

Sidebar Column ဖြစ်ပြီး မိမိသွားလိုရာ page ကို navigate လုပ်နိုင်ရန် နှင့် logout function, user information များကို ပါ၀င်တဲ့ feature ဖြစ်ပါတယ်။

---
## အဓိကလုပ်ဆောင်ချက်များ (Key Features)

### 1. Sidebar Collapsable

- Sidebare ကို hide/visible လုပ်နိုင်ရမယ်။
- Visible ဖြစ်နေချိန်မှာ စာနဲ့ icon မြင်ရပြီး Hide လုပ်ထားချိန်မှာ Icon တခုထဲသာတွေ့နိုင်ရမည်။

### 2. Logged In User Information 

- Logged In ပြုလုပ်ထားသော userရဲ့ information email နှင့် name တကွဖော်ပြပေးနိုင်ရမည်။

### 3. Navigation

- ထည့်ထားသော link များ page သို့ redirect လုပ်ပေးနိုင်ရမည်။
